# Joni Bois A-RT - Store Walkthrough (Final)

The store now features a **Holiday Pop-Up**, a **Brutalist Archive**, and the new **Custom Lab**.

## New Features

### 1. "The Origins" Redesign (Brutalist Theme)
*   **Concept**: A Restricted Vault / Archive.
*   **Visuals**: Concrete texture, heavy typography, "RESTRICTED ACCESS" tape.

### 2. Exclusive Archive "Iced Out" Sale
*   **Event**: 50% Off Winter Sale.
*   **Visuals**: Falling snow, sale badges, "frozen" price tags.

### 3. PROJECT 2026 (Future Drop)
*   **Concept**: High-End / Holographic Teaser.
*   **Visuals**: Glitch text, Countdown Timer to 2026, Blurred "Classified" previews.
*   **Action**: Builds hype for the upcoming January release.

### 4. The Newsletter ("The Inner Circle")
*   **Function**: Email subscription form in the footer.
*   **Style**: Minimalist, dark mode input with neon accent.

### 5. "THE LAB" (Customizer)
*   **Page**: `lab.html` (Accessible via text link in Nav).
*   **Function**: Build your own 1-of-1 prototype.
*   **Workflow**:
    1.  **Select Base**: Switch between White Tee, Black Tee, Hoodie.
    2.  **Select Size**: Choose your fit.
    3.  **Zone Editor**: Click "Chest" or "Arm" to target specific areas.
    4.  **Apply Prefabs**: Click assets (ARs, Logos, Symbols) to stamp them onto the product.

## Preview
> **Hero**: Neon ARs flanking the flickering "Joni Bois" title.
> **Lab**: A tech-interface where you design your own gear.

## How to Run
1.  Open `index.html`.
2.  Scroll down to see the **Project 2026** teaser.
3.  Click **"THE LAB"** in the navigation menu to open the builder.
